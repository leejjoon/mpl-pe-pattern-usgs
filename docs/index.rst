Welcome to mpl_pe_pattern_usgs's documentation!
===================================================================

A package to draw USGS patterns for Matplotlib. The svg files from
`svgpatternusgs <https://github.com/coolbutuseless/svgpatternusgs>`_ (originally from
`geologic-patterns <https://github.com/davenquinn/geologic-patterns>`_), are
processed with `picosvg <https://github.com/googlefonts/picosvg>`_ and converted
to matplotlib's paths.
